WebMCP Demo

This cart exposes its functionality as WebMCP tools via navigator.modelContext.

Open the browser console (F12) and type webmcp.help() to explore, or try: 

webmcp.tools()                                        // list tools
webmcp.call("get-cart-state")                          // read state
webmcp.call("add-item", { name: "Monitor", price: 299 })  // add item
webmcp.call("update-quantity", { itemId: "item-1", action: "increase" })
webmcp.call("remove-item", { itemId: "item-2" })
webmcp.call("clear-cart")
webmcp.listTools()                                     // MCP-compatible JSON
